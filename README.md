Hospital Emergency Room Dashboard -
This Project is an interactive dashboard designed to analyze the performance of a hospital emergency room. It visualizes the key performance indicators according to daily trends over the span of almost two years to help in decision making of hospital management.

Project Overview:
Daily Trend of toatal number of patients coming to hospital,
Age group distribution of daily patient count,
Average waiting time of patients,
Average satisfaction Score,
Gender wise daily patient count,
Percentage of patients admitted,
Department wise referral of patients.

Files included:
Hospital Emergency Room Data.csv (Source Data),
Er_Dashboard.png (Dashboard Screenshot) and
Project_ER HHospital.xlsx (Excel File)

Tools Used
Microsoft Excel,
Pivot Tables,
Pivot Charts,
Slicers,
Power Query, and
Power Pivot.

Purpose:
The goal is to achieve actionable insights from raw data of a hospital emergency room through an interactive exceldashboard in order to help management in decision making.
